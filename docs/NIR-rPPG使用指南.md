# NIR-rPPG 夜间疲劳检测系统使用指南

## 概述

本系统实现了基于近红外(850nm)成像的非接触式心率检测技术，可在完全黑暗的夜间环境下工作。

### 核心创新点

1. **夜间可用**：传统rPPG需要可见光，本系统使用850nm近红外光，夜间可用且不影响驾驶
2. **非接触式**：无需佩戴任何传感器，通过相机即可检测心率
3. **多模态融合**：结合心率、PERCLOS、哈欠、头姿等多个指标，提高可靠性

## 软件配置

### 1. NIR模式配置

已在 `python/config.py` 中配置：

```python
# 相机配置
"is_nir_camera": True,          # 标记为近红外相机
"camera_fps": 30,
"camera_width": 640,
"camera_height": 480,

# rPPG配置
"signal_window_sec": 10,        # NIR模式建议10秒以上
"rppg_method": "NIR_ADV",       # NIR改进算法
"bandpass_low": 0.7,            # 42 BPM
"bandpass_high": 4.0,           # 240 BPM

# HRV配置（NIR模式优化）
"nir_hrv_mode": "simplified",   # 简化模式：仅用心率
"hrv_min_ibi_count": 40,        # 最少IBI数量

# 融合权重（针对NIR优化）
"w_hrv": 0.20,                  # HRV权重降低
"w_perclos": 0.40,              # PERCLOS权重提高
"w_yawn": 0.25,                 # 哈欠权重提高
"w_head": 0.15,                 # 头姿权重
```

### 2. HRV计算模式

NIR模式下提供三种HRV计算模式：

| 模式 | 说明 | 准确度 | 推荐场景 |
|------|------|--------|----------|
| `simplified` | 仅使用心率(HR)，忽略HRV细节 | 高 | **推荐**，最可靠 |
| `trend` | 使用HRV变化趋势 | 中 | 需要长时间监测 |
| `full` | 完整HRV计算 | 低 | 仅用于对比测试 |

**推荐使用 `simplified` 模式**，因为：
- 心率(HR)在NIR模式下准确度可达 ±5-8 BPM
- HRV细节指标(RMSSD/SDNN/LF/HF)在NIR模式下误差较大
- 简化模式更可靠，不会因HRV误差导致误判

## 常见问题

### Q1: 心率检测不准确

**可能原因**：
1. 补光灯亮度不足
2. 面部运动过大
3. ROI区域选择不当

**解决方案**：
```python
# 调整信号窗口长度
"signal_window_sec": 12,  # 从10秒增加到12秒

# 使用鲁棒算法
"rppg_method": "NIR_ROBUST",  # 更好的运动伪影抑制
```

### Q2: HRV风险评分异常

**可能原因**：
- NIR模式下HRV细节指标不准确

**解决方案**：
```python
# 使用简化模式
"nir_hrv_mode": "simplified",  # 仅用心率

# 降低HRV权重
"w_hrv": 0.15,  # 从0.20降到0.15
```

### Q3: 夜间图像太暗

**可能原因**：
- 850nm LED功率不足
- 相机曝光设置不当

**解决方案**：
1. 增加LED数量或功率
2. 调整相机曝光：
```python
# 在 capture/camera.py 中添加
self.cap.set(cv2.CAP_PROP_EXPOSURE, -5)  # 增加曝光
self.cap.set(cv2.CAP_PROP_GAIN, 50)      # 增加增益
```

### Q4: 疲劳等级频繁跳动

**解决方案**：
```python
# 增加平滑系数
"ema_alpha": 0.2,  # 从0.3降到0.2（更平滑）

# 增加持续性约束
"hold_seconds": 8,  # 从5秒增加到8秒
```

## 技术原理

### 为什么NIR-rPPG可行？

1. **血红蛋白的近红外吸收**
   - 血红蛋白在850nm波段仍有光吸收
   - 吸收率约为可见光的30-40%
   - 足以检测心跳引起的血容量变化

2. **单通道强度法**
   - RGB-rPPG利用R/G/B三通道的色度差异
   - NIR-rPPG利用单通道的强度变化
   - 虽然信噪比较低，但通过滤波和平滑可以提取

3. **多区域加权**
   - 额头中央血管最密集，信号最强
   - 太阳穴区域辅助
   - 加权平均提高信噪比

### 算法流程

```
NIR图像序列
    ↓
提取ROI（额头区域）
    ↓
多区域加权平均 → 强度时间序列
    ↓
去趋势 + 标准化
    ↓
带通滤波（0.7-4.0 Hz）
    ↓
BVP信号
    ↓
峰值检测 → IBI序列
    ↓
心率 + HRV指标
    ↓
风险评估
```
