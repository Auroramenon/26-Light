**操作**:

### 第二步：图像采集测试（无补光）

**目的**: 查看相机在无补光情况下的图像

**操作**:

创建测试脚本 `test_camera_raw.py`:

```python
import cv2

cap = cv2.VideoCapture(0)
cap.set(cv2.CAP_PROP_FRAME_WIDTH, 640)
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 480)

print("按 'q' 退出")
print("提示: 如果画面全黑，说明需要850nm补光")

while True:
    ret, frame = cap.read()
    if not ret:
        print("读取失败")
        break
    
    cv2.imshow("Camera Test", frame)
    
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
```

运行:
```bash
python test_camera_raw.py
```

**预期结果**:

- **有滤光片 + 无补光**: 画面全黑（正常，因为滤光片阻挡了可见光）
- **有滤光片 + 有补光**: 画面显示灰度图像，能看到人脸轮廓
- **无滤光片**: 画面显示正常彩色图像

---

### 第三步：850nm补光测试

**目的**: 确认850nm LED补光灯工作正常

**硬件连接**:

1. 将850nm LED补光灯连接到电源
2. 调整补光灯位置，照射到人脸区域
3. 确保补光灯与相机距离适中（30-50cm）

**测试方法**:

**方法A - 肉眼观察**:
- 850nm LED在人眼看来是**微弱的红光**或**几乎不可见**
- 如果看到明亮的红光，可能是普通红光LED（不是850nm）

**方法B - 手机相机测试**:
- 打开手机相机对准850nm LED
- 手机相机能看到**明亮的白光**（手机相机对NIR敏感）
- 这是确认850nm LED工作的最简单方法

**方法C - NIR相机测试**:
```bash
python test_camera_raw.py
```
- 打开补光灯后，画面应从全黑变为灰度图像
- 能清晰看到人脸、手部等物体

## 常见问题排查

### Q1: 画面全黑

**原因**: 850nm滤光片阻挡了可见光，且没有850nm补光

**解决方案**:
1. 打开850nm LED补光灯
2. 调整补光灯位置和亮度
3. 临时测试：移除滤光片，使用室内灯光

### Q2: 检测不到人脸

**可能原因**:
- 补光不足 → 增加补光亮度
- 人脸不在画面中 → 调整位置
- 距离不合适 → 保持30-50cm
- 人脸角度过大 → 正对相机

**调试方法**:
```bash
python test_camera_raw.py
```
查看实际图像质量

### Q3: 心率值异常（<40 或 >120）

**可能原因**:
- 信号采集时移动了 → 保持静止
- 补光不稳定 → 检查电源
- ROI区域不准确 → 调整人脸位置

**解决方案**:
- 重新运行测试
- 增加信号窗口时间（config.py中修改`signal_window_sec`）

### Q4: mediapipe导入失败

**错误信息**:
```
ModuleNotFoundError: No module named 'mediapipe'
```

**解决方案**:
```bash
pip install mediapipe==0.9.3.0
```

**如果网络问题**:
- 参考 `docs/依赖安装问题.md`
- 或临时禁用行为检测，只测试NIR-rPPG核心功能

## 临时测试方案（无850nm补光）

如果暂时没有850nm LED补光灯，可以使用以下方案进行功能测试：

### 方案：移除滤光片 + 室内灯光

**步骤**:
1. 移除相机上的850nm滤光片
2. 在正常室内灯光下测试
3. 修改配置文件:

```python
# config.py
"is_nir_camera": False,  # 改为False
"rppg_method": "CHROM",  # 使用RGB方法
```

4. 运行测试:
```bash
python quick_start_nir.py  # 会自动检测RGB模式
python test_nir_only.py    # 使用CHROM方法
```

**注意**: 这只是临时测试方案，正式使用时必须使用850nm NIR模式。
