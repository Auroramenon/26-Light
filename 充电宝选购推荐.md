# 充电宝选购推荐

> 用途：香橙派 Orange Pi 5 Max（RK3588）疲劳监测演示设备供电
> 场景：现场演示、插电即用、外壳内置 + 外露开关
> 最后更新：2026-05-30

---

## 一、为什么选充电宝

演示原型阶段，**稳定、省心、安全 > 专业感、成本**。对比三种供电方案：

| 方案 | 评价 | 结论 |
|------|------|------|
| **成品充电宝** | 集成稳压+保护+充电，零 DIY 风险，好补给好替换 | ✅ **推荐** |
| 裸 18650 + MT3608 升压 | MT3608 仅 ~2A，喂不饱 RK3588；还缺 BMS 保护 | ❌ 太弱、有安全风险 |
| 树莓派 UPS HAT | 体验好，但 form factor 未必适配香橙派 | ⚠️ 有适配风险，非首选 |

> 裸电池必须额外配「稳压模块 + BMS 保护板 + 充电电路」三件套，等于自己重造一个充电宝，
> 演示阶段得不偿失。除非要可换电池/特殊外形，才考虑 UPS 一体板。

## 二、核心硬指标（缺一不可）

| 指标 | 要求 | 为什么 |
|------|------|--------|
| **5V 档电流** | **必须 5V⎓3A** | RK3588 开机浪涌大，5V/2A 会掉电重启起不来 |
| **持续输出** | 支持「小电流 / 低功耗 / Always On」模式 | 防止程序崩溃空转时被自动断电 |
| **容量** | 10000mAh 左右 | ≈ 2.5~3 小时续航，演示够用 |
| **接口** | 至少一个 PD / 快充 USB 口 | 普通老口往往只有 5V/1A |

> ⚠️ **最大的坑：别只看「30W」。**
> 很多 30W 充电宝的大功率只在 9V/12V 档，**5V 档可能只给 2A**。
> 一定要翻参数表确认 **5V 这一档有 3A**。

## 四、避坑清单

| 方案 | 为什么不选 |
|------|-----------|
| 普通无快充充电宝 | 5V 档常只有 1~2A，扛不住开机浪涌 |
| 裸 18650 + MT3608 升压 | MT3608 ~2A 太弱，喂不饱 RK3588，且无 BMS 保护 |
| 树莓派 UPS HAT | form factor 未必适配香橙派，有适配风险 |
| 只看「30W」不看 5V 档 | 大功率可能只在 9V/12V，5V 仍只有 2A |

## 五、配套件

- **USB 线**：短（≤ 0.3m）、粗，减少压降（Pi 5 Max 对干净 5V 很敏感）
- **开关**：5V 线串一个外露船形开关（KCD11，带红色 LED 指示），GND 直连
- **备用**：演示时多带一块同规格充电宝应急

### 接线示意

```
       充电宝（内置）
       ┌─────┐
       │ 5V  │ ←── 短 USB-A 线（剪开）
       │ 出  │
       └──┬──┘
          │ + ──→ 船形开关（外露）──→ 香橙派 5V Pin
          │ - ─────────────────────→ 香橙派 GND Pin
       └─────┘
```

## 六、到货后必做：冷启动彩排

> 充电宝能不能扛住开机浪涌、会不会自动断电，**只有用真实充电宝冷启动才能验证**。

```
1. 接好线：充电宝 → 开关 → 香橙派
2. 拨开关上电，不碰任何键鼠/屏
3. 掐表观察：
   ✓ 30 秒内小屏亮起 LEVEL 0
   ✓ STM32 灯带显示对应等级
4. 至少彩排 3 次都成功，才算供电方案可靠
```

---

## 七、一句话总结

> 选一块「**5V 档真能给到 3A、支持持续输出**」的品牌快充充电宝，
> 10000mAh，配短粗线 + 外露船形开关。
> 这是踩坑最少、最适合演示原型的供电方案。
